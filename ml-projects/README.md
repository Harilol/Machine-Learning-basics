# House Price Prediction using Machine Learning

# Overview
This project predicts house prices using machine learning techniques based on features such as area, bedrooms, bathrooms, and other amenities.

# Steps Performed
- Data cleaning and preprocessing
- Converted categorical variables using encoding techniques
- Applied StandardScaler for Linear Regression
- Split dataset into training and testing sets

# Models Used
- Linear Regression
- Random Forest Regressor

# Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

# Results
- Linear Regression performed better than Random Forest
- Achieved R² score of approximately 0.64
- Indicates presence of strong linear relationships in the dataset

# Key Learnings
- Importance of data preprocessing
- Handling categorical variables using encoding
- Difference between linear and tree-based models
- Model evaluation and comparison

# Note
Scaling was applied only to Linear Regression, as tree-based models like Random Forest do not require feature scaling.
